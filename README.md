## Deploy Rapidly

1. clone repository

```shell
git clone https://se.jisuanke.com/exam-schedular/ranshaonidemeng/arrange_examination_system.git
```

2. cd ./frontend

```shell
npm install
npm run serve
```

3. open backend with intellij IDEA
